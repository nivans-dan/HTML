Se puede dar un color o características específicas al link, así como colores, subrayado, etc.

En este caso se quiso simular un botón.

Puedes acceder a un link dando click a una imagen.

La ruta absoluta del primer link sería: http://localhost:5500/Lesson04/content.html, sólo se recomienda
usar estas rutas para links externos, no para algo contenido en la misma aplicación.

La ruta relativa del primer link sería: content.html, sólo se usa cuando son extensiones dentro de la misma
aplicación.

Para los links de la misma aplicación se debe poner una diagonal al principio para establecer
la raíz de la ruta, por ejemplo: /css/style.css, es una mejor práctica y lo hace más eficiente.

Al poner dos puntos atrás de una diagonal quiere decir que se mueve un lugar arriba en las carpetas
como sería por ejemplo: ../css/style.css, yendo una carpeta atrás de css (aunque es mejor sólo la '/').
