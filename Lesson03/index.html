Color por código RGB. 255 es el mayor número en la lista.

Color RGB con transparencia.

Color por código HSL (Hue, Saturation, Lightness).

Color HSL con transparencia.

Color por valor Hexadecimal. Se compone de 6 dígitos y por pares.
0 es el menor, f el valor mayor. Como va por pares es como el RGB.
Ej: %00ff00 sería verde porque está en los espacios de Green en RGB.
Igual puede simplificarse a fff ó 000 si se repite en cada par.

Color desde el nuevo archivo CSS (Cascade Style Sheet).
Se llama así porque va desde el elemento padre al elemento hijo.
Ej: de Body a un párrafo, 'Body' es padre y 'p' es hijo.


De esta manera se configuran los márgenes existentes en HTML.

Dejará espacios según lo que nosotros especifiquemos. Podemos hacer esto:
margin(40px 10px 20px 10px) -> (top, right, bottom, left)
De igual manera se puede simplificar como: margin(40px 10px),
siendo (ArribaAbajo, IzqDer).
Si ponemos margin: auto se ajustará automáticamente a 50px.
